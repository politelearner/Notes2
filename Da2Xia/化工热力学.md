# 第5章 相平衡热力学
## 相平衡计算
   **基本平衡式**
   - $x_ip_i^sy_i = py_i\varPhi$
   - $f$平衡
   > 即逸度平衡，列出后计算平衡
   
   **特殊公式**
   - $\frac{G^E}{RT} = \sum x_i\ln(y_i)$
   - Van Laar方程：
     - $\ln(\gamma_1) = A\left[\frac{Bx_2}{Ax_1+Bx_2}\right]^2$
     -  $\ln(\gamma_2) = B\left[\frac{Ax_1}{Ax_1+Bx_2}\right]^2$
  - 


   **共沸点条件**
   - $x_i = y_i$
   - **衍生公式:**
     - $p=p_i^sy_i$
     - $p_1^sy_1=p_2^sy_2$

## 闪蒸计算
   **物料衡算关系式**：
   - $Fz_i = Vy_i + Lx_i$
   - $z_i = ey_i + (1 -e)x_i$
  
   **各种参数**
   1. ``汽化率e`` :$e = \frac{V}{F}$
   2. ``K``: $y_i = K_ix_i$
   3. ``相对挥发度``$\alpha$:$\frac{y_1 / y_2}{x_1 / x_2}$
